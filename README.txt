DESCRIPCIÓN:
El bloque permite conocer la cantidad de tiempo y el número de interacciones de cada usuario del curso durante diferentes tipos de periodo, días, semanas, meses y cursos completos
Se permite la configuración de parámetros de inicio, final y ajustes para hacerlo muy versátil y adaptable a cualquier tipo de curso.
